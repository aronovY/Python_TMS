Домашнее задание:
Необходимо написать конвертор валют, который для расчетов будет использовать апи нацбанка:
http://www.nbrb.by/APIHelp/ExRates
Программа должна принимать три аргумента: цифру, валюту из какой необходимо сконвертировать,
валюту в какую необходимо сконвертировать. Дополнительно программа должна выводить список поддерживаемых валют.
Программа должна устанавливаться в систему как пакет и иметь возможность вызова из терминала.
Дополнительно курсы валюты и историю по запрашиваемым операциям конверсии необходимо хранить в базе данных.
Инструкция по созданию пакетов: https://tehnojam.ru/category/development/sozdaem-svoj-paket-python.html
