Задание на дом:
Необходимо написать программу, которая будет находить все изображения на сайте и скачивать их локально в папку.
Для реализации можно использовать любой подход на ваш вкус - threads, multiprocess, asyncio.
Для парсинга страницы использльзуйте стандартный в python - https://docs.python.org/3/library/html.parser.html

Стоит помнить, что не нужно напускать миллион воркеров на один сайт, это может закончится плохо для всех. 
Лучше всего вынести количество воркеров в отдельную переменную и не привышать 5.
